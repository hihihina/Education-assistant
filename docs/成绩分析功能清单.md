# edua 成绩分析功能清单

## 1. 班级小题分析

- 上传文件：1 份双向细目表 + 1 份班级小题细分数据文件
- 科目识别：优先根据 Excel 文件名自动识别学科名称
- 数据匹配：按题号将班级小题数据与双向细目表自动对应
- 核心指标：
  - 班级整体得分率
  - 薄弱小题数量
  - 薄弱知识点数量
  - 学生总得分率
  - 小题低分占比、零分占比、区分度
- 统计图：
  - 薄弱小题排行图
  - 薄弱知识点排行图
  - 学生总得分率走势 / 排名图
- 明细表：
  - 小题诊断明细表
  - 知识点诊断明细表
  - 学生排名与风险分布表
- 输出目标：
  - 快速定位班级薄弱题目
  - 快速定位班级薄弱知识点
  - 辅助教师进行讲评与二次训练安排

## 2. 年级情况分析

- 上传文件：1 份双向细目表 + 多个班级小题细分数据文件
- 科目识别：优先根据 Excel 文件名自动识别学科名称
- 数据匹配：统一按题号将多个班级文件与同一份双向细目表自动对应
- 核心指标：
  - 班级数量
  - 年级总人数
  - 年级整体均分率
  - 班级整体排名
  - 年级最弱知识点
  - 班级间小题差异
- 统计图：
  - 班级整体排名图
  - 知识点班级热力图
  - 班级差异最大的小题排行图
- 明细表：
  - 班级排名明细表
  - 知识点横向对比表
  - 小题差异排行表
- 输出目标：
  - 比较各班级整体表现
  - 定位各班级薄弱知识点
  - 发现年级层面的共性薄弱点和班级差异点

## 3. 当前规则说明

- 当前版本不提供手工列映射
- 题号命名需与双向细目表保持一致或高度接近
- 双向细目表建议包含：题号、知识点、分值、能力维度
- 结果仅保存在当前浏览会话中，不做持久化
