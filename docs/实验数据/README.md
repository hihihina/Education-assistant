# 实验数据说明

本目录用于测试 `edua` 当前版本的成绩分析模块。

## 文件列表

- `高一数学-双向细目表.xlsx`
  - 用于 `班级小题分析` 和 `年级情况分析`
  - 左侧为题号，右侧为对应知识点
  - 同时补充了分值、能力维度、难度字段，便于做更完整的分析

- `高一1班-数学-小题细分数据.xlsx`
- `高一2班-数学-小题细分数据.xlsx`
- `高一3班-数学-小题细分数据.xlsx`
- `高一4班-数学-小题细分数据.xlsx`
  - 可单独用于 `班级小题分析`
  - 也可多文件一起用于 `年级情况分析`

## 测试方式

- `班级小题分析`
  - 上传 `高一数学-双向细目表.xlsx`
  - 再上传任意 1 个班级小题细分数据文件

- `年级情况分析`
  - 上传 `高一数学-双向细目表.xlsx`
  - 再一次性上传多个班级小题细分数据文件

## 已包含的无效项

班级成绩表中已经故意混入以下无效项目，便于测试解析过滤效果：

- 无关列
  - `家长联系电话`
  - `班主任评语`
  - `德育观察`
  - `备注`
  - `状态`
  - `录入时间`
  - `无效项目`
  - `总分`
  - `平均分`

- 无效单元格
  - 空值
  - `缺考`
  - `待录入`

## 命名说明

- 文件名中包含 `数学`，系统可根据文件名自动识别科目名称
- 班级文件名中包含 `高一1班 / 高一2班 / 高一3班 / 高一4班`，便于识别班级
